## Final submission checklist

- [ ] Source bundle zipped → uploaded to Drive `01_Source_Code/`
- [ ] DOCUMENTATION.md exported as PDF (3 pages or fewer of A4)
- [ ] All 8 screenshots captured at 1280×800+ (mockup + implementation per page)
- [ ] Video uploaded to YouTube **Public**, **comments enabled**
- [ ] YouTube link copied into `04_Video/youtube_link.txt` and into the video
      description on YouTube
- [ ] Drive folder shared with **Anyone with the link → Viewer**
- [ ] Drive link submitted in the LMS / course portal
- [ ] Run instructions verified on a clean machine (or a faculty-owned laptop)
